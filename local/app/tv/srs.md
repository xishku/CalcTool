# 3C电影 华为智慧屏 App — 软件需求规格说明书 (SRS)

## 1. 项目概述

### 1.1 项目名称
3C电影 华为智慧屏客户端（APK）

### 1.2 项目目标
构建一款适配华为智慧屏的 Android TV 应用（APK 格式），聚合 https://www.3ckk.com/ 的影视资源，提供电影、电视剧、微短剧、综艺、动漫五大类内容的浏览与播放功能。应用须完全适配智慧屏遥控器操作（方向键 + 确认键 + 返回键），无需借助鼠标或触摸。

### 1.3 目标用户
- 华为智慧屏 / 荣耀智慧屏家庭用户
- 习惯用遥控器在大屏观看影视内容的用户
- 不想在电视上安装复杂浏览器只想要纯净观影体验的用户

### 1.4 技术栈
- 开发语言：Java / Kotlin
- UI 框架：Android Leanback / Jetpack Compose for TV
- 视频播放：ExoPlayer / Media3（支持 HLS / MP4 / M3U8 流媒体解码）
- 网络层：OkHttp + Jsoup（页面爬取与解析）
- 图片加载：Glide / Coil
- 最低 SDK：Android 7.0 (API 24)，目标 HarmonyOS / Android TV 9+

## 7. HTML 解析方案

### 7.1 解析策略
网站为 SSR 渲染（服务端返回完整 HTML），可直接通过 Jsoup 解析，无需 WebView 执行 JS。

### 7.2 关键解析规则

| 目标 | 解析方式 |
|------|----------|
| 首页推荐 + 列表 | 解析首页 HTML 中的影片卡片 DOM，提取封面、标题、链接 |
| 分类筛选 | 构造分类 URL（如 `/?type=movies&region=国产`）请求并解析 |
| 详情页 | 请求详情页，解析剧情简介、播放列表 DOM |
| 播放地址 | 请求播放页，从 `<script>` 或 `<iframe>` 中提取 M3U8/MP4 真实地址 |
| 搜索 | POST 搜索接口或解析搜索结果页 |

### 7.3 反爬应对
- User-Agent 模拟 TV 浏览器（如 Chrome on Android TV）
- 请求间隔 ≥ 3s
- 解析失败自动重试 2 次
- 不依赖 Cookie/Session

## 8. 遥控器交互规范

### 8.1 按键映射

| 遥控器按键 | 全局行为 | 播放器行为 |
|------------|----------|------------|
| 方向键 ↑ ↓ ← → | 移动焦点 | ← →：快退/快进 10s；↑ ↓：调节音量 |
| OK（确认键） | 进入/选中 | 暂停 / 播放 |
| BACK（返回键） | 返回上一级 | 退出播放，回到详情页 |
| MENU（菜单键） | 打开搜索 / 筛选菜单 | 打开播放设置 |
| 音量键 | 调节系统音量 | 调节音量 |

### 8.2 焦点管理规则
- 页面内焦点循环：到最右后跳到下一行最左，到最下后回到最上
- 页面切换时焦点复位到第一个可聚焦元素
- 弹出层（如筛选、搜索键盘）焦点锁定在层内，BACK 关闭层
- 不可见元素不参与焦点导航
- 焦点动画：250ms 缓动缩放 1.05x + 亮色边框

## 9. 执行流程

### 9.1 应用启动流程
```
1. 展示 Splash 页（Logo + 加载动画）
2. 请求首页 HTML → 解析 HomeData
3. 缓存 HomeData 到本地
4. 进入主界面（默认选中"首页"Tab）
5. 首页数据就绪 → 渲染横行列表
```

### 9.2 观影流程
```
1. 遥控器浏览 → 选中影片 → 按 OK
2. 请求详情页 → 解析 VideoDetail → 展示详情
3. 选中播放线路 → 选中剧集 → 按 OK
4. 请求播放页 → 解析视频流 URL
5. 启动 ExoPlayer 全屏播放
6. 播放中：遥控器控制进度/音量/暂停
7. 退出播放 ← BACK 键 → 回到详情页（自动保存进度）
```

### 9.3 搜索流程
```
1. 按 MENU 键或导航到搜索 Tab
2. 虚拟键盘输入 → 实时联想 / 提交搜索
3. 解析搜索结果列表 → 展示
4. 选中结果 → 进入详情页 → 后续同 9.2
```

## 10. 错误处理

| 场景 | 处理策略 |
|------|----------|
| 无网络连接 | 展示离线提示页，提供"重试"按钮 |
| 网站无法访问（超时/502） | Toast 提示 + 自动重试 2 次，仍失败展示错误页 |
| HTML 解析失败（DOM 结构变更） | 日志记录异常，降级展示"暂无数据"，运营后台告警 |
| 播放地址解析失败 | 自动尝试下一线路，全部失败则提示"该视频暂不可播放" |
| 视频加载超时 | 30s 超时，弹出提示切换线路或重试 |
| 播放器崩溃 | 捕获异常，自动重启播放器并恢复到断点位置 |
| 设备存储不足 | 清理过期缓存（保留最近 7 天），不阻塞正常使用 |
