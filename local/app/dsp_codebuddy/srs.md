1. 业务目标 
提供一个纯GUI操作的工具，为研究员提供从文件导入观察点，并在K线图上快速回看“关注日期”前后行情走势的交互。不要生成样例文件，由用户在GUI上选择路径导入。
________________________________________
2. 数据输入与解析 
选择文件操作：在GUI上提供选择文件菜单，用户点击后指定路径。不要使用命令行方式来让用户输入文件路径。
2.1 文件格式 
•	编码：UTF 8（首选）/ GB18030（兼容）。
•	分隔符：Tab（\t），每行至少2列，最多不超过N列（N受系统内存限制，建议≤100）。
•	字段定义：
1.	datetime：日期时间，不需要任何处理直接显示，其他地方也不会用到，2026-03-30 12:04:08.400673，格式也可能变化，故直接以字符串处理就可以。
2.	symbol：标的代码（必须用字符串存储，如 600000、000011，注意显示到界面不能忽略前面的0，丢失时用0补齐）；
3.	focus_date+：零到多个关注日期（日期格式不同于前面如：20260327，必须用整型存储，调用接口时直接使用，不要再做转换）。
•	行规则：
o	✅ Trim首尾空格后处理；
o	❌ 空行/全空白行跳过；
o	❌ 列数＜2的行记日志并跳过；
o	❌ 日期解析失败的行跳过并告警。
2.2 数据展开 
单行 {dt, sym, fd1, fd2}拆分为独立列表项：
•	[dt, sym, fd1]
•	[dt, sym, fd2]

3. GUI 布局与组件 
3.1 主窗口 
•	最小尺寸：800×600 px，初始建议 1200×700 px。
•	布局：Splitter/BoxLayout 左窄右宽。
区域	宽度	说明
顶部文件栏		
左侧列表	固定 225 px	可横向滚动
右侧画布区	剩余宽度	K线+附图容器（嵌入 ScrollArea）
3.2 左侧列表 
•	表头：日期时间/ 股票代码/ 关注日期
•	排序：按文件原始顺序（后续可扩展点击表头排序）。
•	交互：单选一行 → 触发K线刷新（未加载则请求，已加载则定位焦点日）。
3.3 右侧K线图规格 
•	顶部显示股票的代码/顶部显示股票的代码
•	绘图高度：固定 500 px（不含轴标签外延）。
•	可视区间：默认渲染焦点日 ±25 个交易日（≈50 根）。
•	预加载范围：焦点日前后各100个交易日（共201根）；同标的切换焦点日时复用数据。
•	指标与附图：
o	主图：OHLC K线 + MA(5/10/20/60)；显示阶段的最高/最低价
o	附图1：显示成交量
o	附图2：MACD(12,26,9)。
o	主图和附图的日期用一个就行，不需要重复显示
•	属性显示：每根K线对应 O/H/L/C/V/Amount，交互时 Popup 展示。

4. 交互行为 
4.1 视图操作 
•	缩放：水平缩放K线密度，高度自适应（宽度锁定）。
•	平移/滚动：沿时间轴移动视口，边界平滑缓冲。
4.2 事件响应 
事件	行为	Popup
左键单击K线	设临时焦点日期，显示十字光标，不要显示有遮挡的box	✅ 即时显示该K线数据 更新关注日，同步列表选中	✅ 显示详细指标
右键菜单	ZoomIn/Out、ResetView、ExportPNG	—
Popup内容跟随焦点日期更新（鼠标选择/键盘←/→切换）；弹窗不能超出屏幕边界，最好能拖动位置。

4.3 键盘导航

•	列表模式（无K线选中）：↑/↓ 遍历列表，Enter 加载选中项。
•	图表模式（有K线选中）：
o	←/→：同标的步进交易日（跳过停牌），更新焦点日期，更新Popup；
o	↑/↓：切换相邻列表项（跨标的按需重载）；
o	Esc：取消选中，清除十字线与弹窗。

5.2 性能约束 
•	首次加载（200根+指标）：＜300 ms（本地环境）；

•	内存：单标的数据 ≤50 KB，缓存最近5标的。

6. 边界与异常 
•	非交易日/停牌：K线留白，Popup 提示“无数据”；
•	极值标尺：自动适配价格轴，保持连续性；
•	币种单位：跟随市场（CNY/USD），无自动换算。

12. 实现约束（强制要求） 
12.1 架构与编码 
•	Class封装：核心逻辑必须用类隔离，禁止全局函数堆砌；
•	单一交付文件：所有类置于同一 .py文件，模块间用注释分块；
•	代码规范：
o	函数≤50行，行宽≤120字符；
o	snake_case（变量/函数），CamelCase（类），常量全大写；
o	禁魔数，参数化配置。
12.2 日志与错误 
•	统一日志接口：使用 SDK Logger，禁用 print；import出错直接报错，不要使用模拟的ImportError

具体样例及导入方法：
print(os.path.join(os.path.dirname(os.path.realpath(__file__)), "../../../src"))
sys.path.append(os.path.join(os.path.dirname(os.path.realpath(__file__)), "../../../src"))
from CalcTool.sdk.tool_main import CalcLast1YearCount
from CalcTool.sdk.logger import Logger

Logger.log().info(“xxx”)

o	INFO：文件行数、请求起止；
o	WARN：无效行、部分缺失；
o	ERROR：超时、解析失败；
不要对日志接口再次封装，这样导致出现问题看不到具体的代码行：
    def log_info(msg): Logger.log().info(msg)
    def log_warn(msg): Logger.log().warn(msg)
    def log_error(msg): Logger.log().error(msg)

•	GUI友好反馈：非阻塞提示，异常不崩，保留前一有效状态。
12.3 资源与安全 
•	缓存上限5标的（LRU淘汰）；
•	耗时IO异步化，GUI线程不锁；
•	输入消毒（路径/参数校验）。
